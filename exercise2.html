Exercise 2

Create a CF application that contains a form in which the user enters their details including name, DOB, house number and post code. The form will contain a button that when clicked, will find the full address via an ajax call. If the address is found, the address fields, which are not shown by default, will be shown on the form populated with the correct data. If the address is not found, a message is displayed on the form. (The valid addresses will be stored in memory on the CF server). Note: Finding and populating the address should not refresh the page
